Dansk TV

The source code for the Dansk TV app is available at https://github.com/mkjensen/tv.

Privacy policy

No information is collected by the developer of Dansk TV.

Dansk TV uses the following third-party tools to enable crash reporting. Links to the respective privacy policies:

  • Crashlytics
  • Fabric
  • Firebase

Dansk TV provides a user interface for content provided by Danish Broadcasting Corporation.
